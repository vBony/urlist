# urlist

  Urlist é um sistema para armazenar e listar seus contatos, desenvolvido em PHP 7.2, JS, Jquery, HTML5/Css3 e Bootstrap 4
  
  Este é um projeto didático, onde tudo que aprendo é implementado aqui (se possível). Não estou focando tanto no frontend,
mas de vez em quando eu realizo faço alguns updates.

  Atualmente o urlist não se encontra em versão final, mas falta apenas alguns ajustes e funcionalidades para isso.
  
  Daqui uns dias publicarei algumas fotos e vídeos do projeto, e talvez irei hospeda-lo.
